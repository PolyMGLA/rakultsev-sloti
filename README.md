# Ракульцев-казино

### Usage

- Установите Redis: ```sudo pacman -S redis```, ```sudo apt-get install redis-server``` или другая команда в зависимости от вашей системы
- Проверьте состояние Redis: ```redis-cli ping``` - ответ должен быть ```PONG```
- Установите зависимости: ```pip install -r requirements.txt```
- Настройте файл .env
- Запустите шедевробота: ```python main.py```

### Фичи

- Слоты
  * топ лист
  * возможность крутить слоты
  * система секретных комбинаций
  * система додепов
- Блэкджек
- Магазин
  * кейсы
